# Extract Text In Between
## Run
```
python main.py
```
## Instruction
This script capture the text in bewteen of the FIRST " and the SECOND " in each line of the txt file (`input.txt`).
Free to modify the codes catering to your need, by changing these two lines:
```
startID = line.find('\"')
```
and
```
endID = newline.find('\"')
```
> Note: The order of the character is important.
